# 🫀 Heart Disease Risk Prediction

## 📌 Overview
This repository contains a machine learning project designed to predict the likelihood of a patient developing heart disease based on clinical parameters. The project explores data preprocessing, exploratory data analysis (EDA), and compares the performance of multiple classification algorithms to identify the most accurate predictive approach for healthcare applications.

## 🛠️ Tech Stack & Libraries
* **Language:** Python
* **Data Manipulation:** Pandas, NumPy
* **Machine Learning:** Scikit-Learn, XGBoost
* **Data Visualization:** Matplotlib, Seaborn

## 🚀 Key Features & Workflow
1. **Exploratory Data Analysis (EDA):** Analyzed clinical health indicators and visualized data distributions to uncover hidden patterns.
2. **Data Preprocessing:** Handled missing values, encoded categorical variables, and scaled numerical features for optimal model training.
3. **Model Training & Comparison:** Developed and evaluated three classification models:
   * Logistic Regression
   * Random Forest Classifier
   * XGBoost Classifier
4. **Model Evaluation:** Assessed models based on accuracy, precision, recall, and F1-score to ensure high reliability in a medical context.
